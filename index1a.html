Context

After identifying the 4 cowrie divination system embedded within Merindilogun, the efficiency of the 20-4 system becomes visible when applying Elderization as the corrective technology.

When Eldership is respected, everything is in Ire. When Eldership is violated, Ibi arises and Eldership seeks to be re-established.

The diagram above illustrates the flow of comparisons: Eldest → Elder → Senior → Junior (and back), until Eldership is restored.

Elderization Protocol (20-4)

  1. If Eldest–Elder shows disrespect, Eldest is in Ibi.
  2. Elderize by comparing Eldest → Senior.
  3. If still unclear, compare Eldest → Junior.
  4. If still unresolved, compare Elder → Senior.
  5. Then Elder → Junior.
  6. Finally Senior → Junior.
  7. If no pairing restores Eldership, cast again.

Daily Assignment

Cast four times until Ire is established. Document all Ire / Ibi sequences.

Practice for 8 consecutive days. Restart if ANY day is missed.
